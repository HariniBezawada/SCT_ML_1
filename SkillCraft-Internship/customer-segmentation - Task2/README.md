# Customer Segmentation using K-Means Clustering

## Objective
Group retail store customers based on purchasing behavior using K-Means Clustering.

## Dataset
Mall Customers Dataset

## Features Used
- Annual Income (k$)
- Spending Score (1-100)

## Technologies
- Python
- Pandas
- Matplotlib
- Scikit-Learn

## Algorithm
K-Means Clustering

## Outputs
- Elbow Method Graph
- Customer Cluster Visualization
- Cluster Summary Report

## Run

pip install -r requirements.txt

python customer_segmentation.py
