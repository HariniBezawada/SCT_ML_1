# Hand Gesture Recognition

## Objective

Develop a hand gesture recognition model that identifies and classifies different hand gestures from real-time webcam video.

## Features

- Real-time hand detection
- Hand landmark tracking
- Detects:
  - Fist
  - Palm
  - One
  - Two
  - Three
  - Four
- Gesture classification
- Live webcam interface

## Technologies Used

- Python
- OpenCV
- MediaPipe
- NumPy

## Installation

Install dependencies:

pip install -r requirements.txt

## Run

python hand_gesture.py

## Controls

Press ESC to exit.

## Applications

- Human Computer Interaction
- Gesture-based control systems
- Sign language basics
- Virtual interfaces
